# Project Stack

git-lui uses the following tools (and of course, ghc):

- [cabal]: build system
- [hpack]: DRY package declaration
- [haddock]: inline documentation
- [tasty]: testing meta-framework
- [doctest]: test haddock code examples
- [weeder]: dead code elimination
- [hlint]: linting
- [stan]: deep static analysis linting

[cabal]: https://www.haskell.org/cabal/
[doctest]: https://hackage.haskell.org/package/doctest
[haddock]: https://haskell-haddock.readthedocs.io/latest/
[hlint]: https://github.com/ndmitchell/hlint
[hpack]: https://github.com/sol/hpack
[stan]: https://github.com/kowainik/stan
[tasty]: https://github.com/UnkindPartition/tasty
[weeder]: https://github.com/ocharles/weeder
